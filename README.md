# Pomodoro
Pomodoro em node.js

Pomodoro CLI

Um simples e eficiente Pomodoro no terminal, feito com Node.js.
Ele ajuda a melhorar a produtividade usando o método Pomodoro (ciclos de foco e pausa diretamente no terminal, sem interface gráfica).

O projeto é leve, rápido e ideal para quem quer manter foco durante estudos ou trabalho sem distrações.

🚀 Funcionalidades
⏱️ Timer de foco (25 minutos)
☕ Intervalo de descanso (5 minutos)
🖥️ Interface no terminal (CLI)
🔁 Execução contínua de ciclos
⚡ Leve e sem dependências pesadas
Tecnologias usadas
Node.js
JavaScript (ES6)
Terminal (CLI)
